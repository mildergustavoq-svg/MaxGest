--- MaxGest ---

Descripcion:

    Los presentes scrips son parte de un solo programa realizado con python, su función principal es ayudar a gestionar
    pequeños emprendedores quieren poner su negocio propio y muchos fallan. Por ello, presentamos una aplicación diseñada para ayudar a los emprendedores a administrar sus empresas de forma óptima.
    Muchos negocios locales pierden dinero al no saber con precisión qué mercancía tienen disponible, lo que genera compras duplicadas o quiebres de inventario, con el fin de ayudar a estas personas se creo esta version de modelo de gestion de negocio llamado "MaxGest".

Funcionalidades:

    El presente trabajo final al cual nos referiremos por su nombre "MaxGest" es capaz de cumplir varias funciones para un asistente
    de gestión básico:
        - Almacenar datos de los productos.
        - Simular ventas para registrar las cantidades.
        - Realizar predicciones básicas según el comportamiento de ventas.
        - Eliminar producto cuando el usuario lo desee.
        - Editar el producto cuando el usuario lo desee.
        - Presentar que productos tienen un stock bajo.
        - Presenta un historial de transacciones.
        - Resume de manera gráfica las ventas por hora en el rango de un día.
        - Resume de manera gráfica las ventas por dia en el rango de un Mes.
        - Contabiliza la cantidad de productos registrados.
        - Contabiliza la cantidad de items totales.
        - Registra el total de venta registradas diarias.
        - Registra el total de venta mensuales registradas hasta el momento.
        - Busca los productos si el usuario tiene una gran cantidad de productos.
        - Registra el tiempo de cada transacción realizada.
